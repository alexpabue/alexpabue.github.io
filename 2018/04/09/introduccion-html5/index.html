Introducción a HTML5.

HTML es un lenguaje de marcado que se utiliza para la creación de páginas web. Para crear un documento html únicamente necesitamos un editor de texto y guardar el documento con la extensión html.

Estructura básica

La etiqueta <!DOCTYPE html>se utiliza para indicar al navegador web que va a interpretar el contenido de la página web que ha sido escrita utilizando html en su versión 5.

Las etiquetas <html> de apertura y </html> de cierre indican que cualquier contenido incluido entre estas etiquetas es contenido html.

La etiqueta <head> </head> es una etiqueta que contiene información sobre la página. Dentro de la etiqueta <head></head> tenemos la etiqueta <title></title> que es la etiqueta que se utiliza para dar título a la página web. Es el título que vemos en la parte superior del navegador web:

La etiqueta <body></body> es la que se utiliza para mostrar el contenido en la ventana del navegador. Cualquier cosa escrita entre las etiquetas <body></body> se verán en la ventana del navegador:

Crear una página sencilla

Para crear un documento html únicamente necesitamos un editor de texto y pegar el código html. Utilizaremos Sublime Text.
